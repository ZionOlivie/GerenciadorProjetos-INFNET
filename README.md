# Gerenciador de Projetos (CRUD) 📊

Sistema web desenvolvido em C# e ASP.NET Core (Razor Pages) para o gerenciamento de Projetos, tarefas e membros de equipe. Projeto de bloco construído para a faculdade INFNET
Focando na aplicação prática de modelagem de dados e Razor

## 🚀 Funcionalidades
* **Gestão de Usuários:** Cadastro de membros e definição de perfis (Gerente ou Membro).
* **Gestão de Projetos:** Criação de projetos com atribuição de um Gerente responsável e alocação de múltiplos membros na equipe.
* **Gestão de Tarefas:** Criação de tarefas vinculadas a projetos específicos e atribuição de responsabilidade a membros individuais.
* **Interface Dinâmica:** Telas interativas que exibem os dados cruzados (ex: listar a equipe completa na página de detalhes do projeto).

## 🛠️ Tecnologias Utilizadas
* **Back-end:** C# .NET 8.0
* **Front-end:** HTML5, CSS3, Bootstrap 5 e Razor Pages
* **Banco de Dados:** SQLite (para desenvolvimento rápido)
* **ORM:** Entity Framework Core (Code-First & Eager Loading)
